# Project conventions

## Language

This project is **English-only, everywhere, no exceptions**:

- Source code: identifiers, comments, doc comments, log messages, error
  strings, CLI output.
- Tests: names, fixtures, assertion messages.
- Git: commit messages, branch names, PR titles and descriptions.
- Docs: README, inline help, this file.
- Shell scripts and CI workflows: comments and echoed strings.

No French (or any other language) is allowed in commits. If you spot any,
fix it in the same change.
